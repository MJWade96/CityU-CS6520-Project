## 🎯 与原计划对比

### 原计划：StatPearls
- 9,330 篇文章
- 301,202 个文本片段
- 平均 119 tokens/片段
- 需要从 NCBI Bookshelf 下载

### 实际采用：MedRAG/textbooks ✅
- 18 本权威教科书
- 125,847 个文本片段
- 平均约 777 字符/片段
- 直接从 Hugging Face 下载

### 优势
1. **更易获取** - 无需注册，公开可用
2. **更权威** - 经典医学教科书
3. **覆盖全面** - 基础科学 + 临床医学 + 考试复习
4. **已分块** - MedRAG 已处理好的段落级分块
5. **格式兼容** - 与现有 RAG 系统完全兼容

## 📝 数据格式

每个文本片段包含以下字段：

```json
{
  "id": "Anatomy_Gray_0",
  "title": "Anatomy_Gray",
  "content": "What is anatomy? Anatomy includes...",
  "contents": "Anatomy_Gray. What is anatomy?...",
  "source": "medrag_textbooks",
  "textbook": "Anatomy_Gray"
}
```
