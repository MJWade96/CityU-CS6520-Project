## 三、各模块详细配置

### 模块对比表

| 模块 | 方法/模型 | 是否使用LLM | 说明 |
|------|-----------|-------------|------|
| **数据层** | HuggingFace datasets | ❌ 否 | 本地数据加载 |
| **索引模块** | sentence-transformers | ❌ 否 | 本地嵌入模型 |
| **向量存储** | FAISS | ❌ 否 | 本地向量数据库 |
| **检索模块** | Dense Retrieval | ❌ 否 | 向量相似度搜索 |
| **生成器** | **Qwen2.5-7B** | ✅ **是** | **唯一使用LLM** |
| **评估模块** | Rule-based | ❌ 否 | 规则+统计方法 |

---

### 3.1 索引模块配置

```python
# 嵌入模型配置 (不使用LLM)
embedding_config = {
    "model": "sentence-transformers/all-MiniLM-L6-v2",
    "dimension": 384,
    "device": "cpu",  # 或 "cuda"
    "normalize": True
}

# 文本分割配置 (不使用LLM)
splitter_config = {
    "chunk_size": 512,
    "chunk_overlap": 50,
    "separators": ["\n\n", "\n", ". ", " ", ""]
}
```

**可选嵌入模型**：

| 模型 | 维度 | 速度 | 适用场景 |
|------|------|------|----------|
| all-MiniLM-L6-v2 | 384 | 快 | 通用场景 |
| all-mpnet-base-v2 | 768 | 中 | 高质量需求 |
| paraphrase-multilingual-MiniLM-L12-v2 | 384 | 中 | 多语言 |
| emilyalsentzer/Bio_ClinicalBERT | 768 | 慢 | 医学专业 |

---

### 3.2 向量存储配置

```python
# FAISS配置 (不使用LLM)
vector_store_config = {
    "type": "FAISS",
    "metric": "cosine",  # 余弦相似度
    "dimension": 384,    # 与嵌入模型匹配
    "storage": "memory", # 或 "disk"
    "index_type": "IndexFlatIP"  # 内积索引
}
```

---

### 3.3 检索模块配置

```python
# 检索配置 (不使用LLM)
retrieval_config = {
    "method": "dense_retrieval",
    "top_k": 5,
    "similarity_threshold": 0.7,
    "reranking": False  # 可选：使用cross-encoder重排序
}
```

---

### 3.4 生成器模块配置 (唯一使用LLM)

```python
# Qwen2.5-7B配置 (唯一使用LLM的模块)
generator_config = {
    "model": "qwen2.5:7b",
    "deployment": "Ollama",
    "base_url": "http://localhost:11434",
    "temperature": 0.1,
    "max_tokens": 1024,
    "top_p": 0.9
}
```

**Qwen2.5-7B部署方式**：

| 方式 | 命令 | 说明 |
|------|------|------|
| **Ollama** | `ollama pull qwen2.5:7b` | 推荐，简单易用 |
| **vLLM** | `python -m vllm.entrypoints.api_server` | 高性能推理 |
| **Transformers** | `AutoModelForCausalLM` | 直接加载模型 |

## 六、部署指南

### 6.1 安装Ollama并下载Qwen2.5-7B

```bash
# 1. 安装Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 2. 启动Ollama服务
ollama serve

# 3. 下载Qwen2.5-7B
ollama pull qwen2.5:7b

# 4. 验证安装
ollama list
# 应该看到: qwen2.5:7b
```

### 6.2 安装Python依赖

```bash
pip install langchain langchain-community langchain-core
pip install langchain-text-splitters
pip install faiss-cpu
pip install sentence-transformers
pip install datasets
pip install fastapi uvicorn
```

### 6.3 运行系统

```python
from app.rag.qwen_medical_rag import MedicalRAGSystem, MedicalRAGConfig

# 配置
config = MedicalRAGConfig(
    llm_model="qwen2.5:7b",
    llm_base_url="http://localhost:11434",
    embedding_model="sentence-transformers/all-MiniLM-L6-v2"
)

# 初始化
system = MedicalRAGSystem(config)
system.initialize()

# 查询
result = system.query("高血压的一线治疗方案是什么？")
print(result['answer'])
```

## 七、与原架构对比

| 特性 | 原架构 | 新架构 (Qwen2.5-7B) |
|------|--------|---------------------|
| **嵌入模型** | OpenAI API | sentence-transformers (本地) |
| **向量存储** | FAISS | FAISS (不变) |
| **LLM** | GPT-4o-mini (API) | Qwen2.5-7B (本地) |
| **评估方法** | RAGAS (需要LLM) | Rule-based (不需要LLM) |
| **LLM使用范围** | 嵌入+生成+评估 | **仅生成器** |
| **部署方式** | 云API | 完全本地 |
| **成本** | 按使用付费 | 免费 |

## 八、优势总结

1. **LLM使用最小化**: 仅在生成器模块使用LLM
2. **完全本地部署**: 无需API密钥，无网络依赖
3. **成本为零**: Qwen2.5-7B免费使用
4. **隐私保护**: 数据不离开本地
5. **可控性强**: 完全控制所有组件
6. **中文优化**: Qwen2.5对中文支持优秀
