### 合并语料库格式

每个片段包含：
```json
{
  "id": "Anatomy_Gray_0",
  "title": "Anatomy_Gray",
  "content": "What is anatomy?...",
  "contents": "Anatomy_Gray. What is anatomy?...",
  "source": "medrag_textbooks",
  "textbook": "Anatomy_Gray"
}
```

PubMed 片段额外包含：
```json
{
  "source": "pubmed",
  "pmid": "12345678",
  "journal": "Journal Name",
  "year": "2024",
  "authors": ["Author1", "Author2"]
}
```
